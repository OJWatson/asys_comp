What this site is for

If you are new to this work: this is the front door for the ASYS screening programme. Each project gets one deep-dive page with plain-language context, methods, and decision support. Use this home page to choose a project; use project pages to understand screening choices; use Shared LAB to run day-to-day review.

How to read this site

  1. If you are new, read the Handbook first for the screening workflow and decision rules.
  2. Then open the project you need from Projects below.
  3. Read the project’s Overview and Recommendation first.
  4. Use the project’s Methods, Risk, and Planner sections only when you need detail.
