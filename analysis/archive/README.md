# Analysis Archive

This folder stores non-canonical historical planning/progress material and dated execution snapshots.

## Contents

- `reports/` dated generated reports/checklists retained for audit trail
- `*_2026-02-21.md` planning/progress documents from prior implementation phases

## Canonical active artifacts remain in

- `analysis/outputs/`
- `analysis/ASREVIEW_IMPROVEMENT_REPORT.md`
- `analysis/DEPLOYMENT_RUNBOOK.md`

Historical outputs that are not needed for current site/tests should be moved here or into
an ignored `analysis/outputs/**/archive/` path.
